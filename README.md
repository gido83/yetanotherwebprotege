# YetAnotherWebProtege App

A new product direction for a simpler, more approachable ontology editor.

This project is intentionally separate from the legacy WebProtege checkout. Use the cloned `protegeproject/webprotege` repository as a reference for ontology behavior, import/export, permissions, collaboration, and data model decisions. Build the new user experience here.

## Goals

- Easy to use for non-expert ontology contributors.
- Short learning curve with guided workflows and plain-language copy.
- Visually clear, modern, and focused on the user's next action.

## Development

```bash
npm install
npm run dev
```

The app runs at `http://127.0.0.1:5173`.
